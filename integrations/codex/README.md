# Codex

OpenAI Codex CLI reads `AGENTS.md` files for project-local agent guidance.

1. Copy `AGENTS.md` to the root of your project (or merge its content into
   an existing `AGENTS.md`).
2. Ensure `webfetch` is on `$PATH` (run `install/install.sh`).
3. Codex will reference the rules automatically when working in the project.
