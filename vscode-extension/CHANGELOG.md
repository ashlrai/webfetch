# Change Log

## 0.1.0 — Initial release

- Activity-bar side panel with federated search across 19 providers.
- License color badges, provider chips, license-policy filter.
- Click to download + insert; drag to drop into any editor.
- Right-click in markdown to insert a licensed image with attribution.
- Command palette: Search, Providers, Set API Key, Open Dashboard, Insert image.
- Status-bar indicator reflecting connection state.
- Inline markdown quick-fix that suggests a webfetch search when typing `![alt]()`.
- Optional XMP sidecar written next to each downloaded image.
- Settings: API key (in SecretStorage), base URL, default license, default providers, output directory, attribution style, sidecar toggle.
